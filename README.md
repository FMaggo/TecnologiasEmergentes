Title: Image Transformation

IMPORTANT: Read Wiki page on Github for further disclosure of project.

Description: Code used to tranform images in different manner (i.e: Grey scale, flipped horizontal- vertically, Blur via Kernels)

Installation instructions: 
1. Have OpenMP in device
2. Download "image_transforms_omp.c
3. Download images to transform

Features or functionalities: Transforms three images of 2000x2000 pixels via Flipping, grey-scaling and blur.

Configuration options: Number of threads to use.
